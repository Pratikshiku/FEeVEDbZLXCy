### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 车位信息
5. 社区信息
6. 周边服务
7. 问卷调查
8. 爱心助老
9. 通知公告
10. 留言反馈
11. 个人中心
12. 我的收藏
13. 客服中心


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 客户管理
4. 在线报修管理
5. 投诉建议管理
6. 车位信息管理
7. 车位租买管理
8. 社区信息管理
9. 周边服务管理
10. 问卷调查管理
11. 参与答卷管理
12. 爱心助老管理
13. 留言板管理
14. 系统管理（客服中心、轮播图管理、通知公告）
